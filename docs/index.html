What it does

🎛️ Manage many serversOne dashboard, a start/stop switch per server, live status at a glance.
📥 One-click setupCreates the folder, downloads the server jar (Vanilla, Paper, Purpur, Fabric), and even downloads Java for you.
🧩 Built-in mod & plugin browserSearch and install from Modrinth, filtered to what your server can actually run.
⚡ Quick commandsFriendly fill-in-the-blank admin actions with player-name autocomplete — no syntax to learn.
🌍 Easy sharingShows your local & public address with copy buttons and a plain-English port-forwarding guide.
👥 Players, files & moreOP toggles, gamemode, a file editor, server.properties form, RAM sliders, EULA helper, and a guided tour.

Getting started

  1. Download for your OS above and open it.
  2. Click ＋ to add a server — the guided tour walks you through it.
  3. Pick your server software, set RAM, and accept the Minecraft EULA.
  4. Flip the toggle to start. Share your address from the Connect tab. That's it!

VoxelDeck needs Java to run servers — it can download the newest Java for you automatically, or you can install it yourself.

Good to know

Unsigned builds. These downloads aren't code-signed yet, so Windows SmartScreen may say "unknown publisher" and macOS may call it "unidentified" — that's expected for an indie app. On Windows click More info → Run anyway; on macOS right-click the app → Open. VoxelDeck only manages servers on your computer and never touches your Minecraft account. Not affiliated with Mojang or Microsoft.
